# TakeAway Restaurant Management System (Java)

## Description
This is a desktop-based restaurant management system developed using Java and NetBeans GUI. It allows management of customers, orders, and menu items.

## Features
- Customer management
- Order processing
- Menu management
- GUI-based interaction

## Technologies
- Java
- NetBeans GUI
- MySQL

## Structure
- Customer class
- Order class
- MenuItem class
- OrderItem class
