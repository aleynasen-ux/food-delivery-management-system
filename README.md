# Food Delivery Management System (Java)

## Description
This is a desktop-based restaurant management system developed using Java and NetBeans GUI. It allows management of customers, orders, and menu items.

## Features
- Customer management
- Order processing
- Menu management
- GUI-based interaction

## Technologies
- Java
- NetBeans GUI
- MySQL

## Structure
- Customer class
- Order class
- MenuItem class
- OrderItem class
  
## Database Setup
- Import the SQL file into phpMyAdmin or MySQL
- Update DBConnection.java with your credentials

## How to Run
1. Open project in NetBeans
2. Import SQL file into MySQL
3. Update DBConnection.java
4. Run the project
